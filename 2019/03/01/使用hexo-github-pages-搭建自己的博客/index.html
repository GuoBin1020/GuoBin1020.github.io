使用hexo+github pages 搭建自己的博客

发表于 2019-03-01 | 更新于 2019-08-27 | 分类于 博客搭建 | 评论数： | 热度： ℃
这是一篇有关如何使用hexo+github pages搭建自己独立博客的教程，也是基于其他博主的博客内容做的（嗯，站在巨人的肩膀上），也会记录一些在搭建的过程中碰到的问题。

前言

作为一枚程序员，终于在最近决定搭建自己的博客，希望在以后养成写博客的好习惯，总结一些技术上的问题以及积累，有感兴趣的小伙伴可以跟随本文搭建自己的博客，有没有很鸡冻呢，接着往下看吧。

准备工作

在一些开始之前，你首先需要准备：

  • 一个github的账号，没有的话就需要注册。
  • 安装node.js、npm，新版本的node.js自带npm，node.js下载，本文使用node.js版本为10.15.2，npm版本为6.4.1。
  • 安装git for windows，或者其他git客户端，git for windows下载，本文使用git for windows版本为2.21.0。
