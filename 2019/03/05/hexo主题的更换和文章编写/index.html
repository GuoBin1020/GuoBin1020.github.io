hexo主题的更换和文章编写

发表于 2019-03-05 | 更新于 2019-09-26 | 分类于 博客搭建 | 评论数： | 热度： ℃
个人博客搭建好了之后怎么让它变的更加美观？使用markdown语法编写文章轻松克服排版问题。

前言

使用hexo+github pages成功的搭建了一个简单的个人博客，那么，怎么让这个站点更加的美观，功能更加的强大呢？答案就是换一个强大的主题。hexo-themes-next这个主题是一个非常热门的主题，本文将更换主题的过程记录下来，方便日后参考。同时记录一些简单的markdown语法。

准备工作

  • 安装git for windows，或者其他git客户端，git for windows下载，本文使用git for windows版本为2.21.0。
  • markdown编写工具，在线的也行，比如小书匠
